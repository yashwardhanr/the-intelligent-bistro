## 📌 Project Summary

BistroMind AI is an end-to-end restaurant ordering experience built with a **React Native Expo mobile app** and a **Node.js Express backend**.

The app allows users to browse a bistro-style menu, add dishes manually, manage a shopping cart, and use an AI assistant to place or modify orders conversationally.

The main focus of the project is that the AI does not simply respond with text. It interprets user intent and returns **structured JSON actions** that are applied directly to the cart.
